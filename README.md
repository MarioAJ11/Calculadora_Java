# Calculadora Java

Calculadora muy simple, entrada y salida por terminal. Simplemente, reforzar uso de lenguajes, buenas practicas y comentarios utiles. Anticipacion de errores de usario.

---

## Tecnologías Utilizadas

- Java: JDK 21

## Funcionalidades

- Funciones

    1. Sumar
    2. Restar
    3. Multiplicar
    4. Dividir
    5. Potencias

- Se controla

    1. Entrada equivoca de String en vez de numeros.
    2. Salida inteligente de datos.
    3. Limpieza de terminal.
    4. Buenas practicas como el cierra de Scanner.

- Interfaz

    1. Menu basico por terminal.
    2. Menu infinito hasta que la opcion sea 0.

## Cómo Ejecutar

1. Descargar el .zip o usar ```git clone https://github.com/MarioAJ11/Calculadora_Java.git``` en el directorio deseado.
2. Abrir la terminal y movernos usando ```cd```.
3. Para ejecutar el programa primero hay que compilarlo con ```javac -cp src\*.java```.
4. Luego para iniciarlo ```java -cp src Main```

La salida por terminal seria algo asi.

![imagen terminal](src/image.png)

Ya se usaria el programa introduciendo numeros por terminal, los decimales con ```,```.
